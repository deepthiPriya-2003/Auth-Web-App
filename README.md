# Auth-web-App

## Overview

This is a responsive Dashboard built using React.js.
It displays and manages data for Tasks, Leads, and Users with a clean UI and modern design.
The application simulates real-world API behavior by fetching data from a local JSON file.

### Tech Stack
   React.js
   CSS (Flexbox & Responsive Design)
   Fetch API
   JSON (Mock Data)

### Features 
  Dedicated Login page with valid authentication using form inputs (username, email, password) with proper error handling.
  Dynamic data rendering (Tasks, Leads, Users)
  API simulation using Fetch
  Modern UI with gradient backgrounds
  Card-based layout
  Responsive design
  Clean and reusable code structure


### Setup & Installation

1.Clone the repository

2.Navigate to project folder

3.Install dependencies
npm install

4.Run the app
npm start
